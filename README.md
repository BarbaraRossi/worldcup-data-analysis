# worldcup-data-analysis
Análise exploratória de dados das Copas do Mundo da FIFA

## Estrutura inicial do projeto

- `1-extracao/`: datasets brutos e documentação da etapa de extração.
- `2-limpeza/`: etapa reservada para tratamento e padronização dos dados.
- `3-analise/`: etapa reservada para análises exploratórias e analíticas.
- `4-visualizacao/`: etapa reservada para visualizações e comunicação de resultados.
